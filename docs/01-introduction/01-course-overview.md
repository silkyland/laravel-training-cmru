# 1.1 Course Overview (ภาพรวมหลักสูตร)

> **บทนี้คุณจะได้เรียนรู้**
> - เป้าหมายและวัตถุประสงค์ของหลักสูตร
> - กำหนดการอบรม 2 วัน
> - เครื่องมือที่ต้องใช้ในการอบรม
> - การใช้ AI ในการพัฒนา Laravel

---

## วัตถุประสงค์การเรียนรู้

เมื่อจบบทเรียนนี้ ผู้เรียนจะสามารถ:
1. อธิบายเป้าหมายและโครงสร้างของหลักสูตรได้
2. เข้าใจกำหนดการและหัวข้อที่จะได้เรียนรู้ตลอดการอบรม
3. เตรียมเครื่องมือที่จำเป็นสำหรับการพัฒนา Laravel ได้ครบถ้วน
4. เข้าใจแนวคิดการใช้ AI เป็นผู้ช่วยในการพัฒนา

---

## เนื้อหา

### 1. เป้าหมายของหลักสูตร

หลักสูตรนี้มุ่งเน้นให้ผู้เข้าอบรมสามารถพัฒนาเว็บแอปพลิเคชันด้วย **Laravel Framework** โดยใช้ **AI (Artificial Intelligence)** เป็นผู้ช่วยในการเขียนโค้ด เพื่อให้การพัฒนารวดเร็ว แม่นยำ และปลอดภัยตามมาตรฐานสากล

### 2. กำหนดการอบรม

#### วันที่ 1: พื้นฐานและการจัดการข้อมูล

| ช่วงเวลา | หัวข้อ | รายละเอียด |
|---------|-------|------------|
| **เช้า** | แนะนำ Laravel | ประวัติ, สถาปัตยกรรม, การติดตั้ง |
| | Routing | การกำหนดเส้นทาง URL และ Parameters |
| | Controllers | การจัดการ Request และ Response |
| **บ่าย** | Eloquent ORM | การจัดการฐานข้อมูลแบบ Object-Oriented |
| | Blade Templates | การสร้าง Views และ Components |

#### วันที่ 2: ความปลอดภัยและการใช้งานจริง

| ช่วงเวลา | หัวข้อ | รายละเอียด |
|---------|-------|------------|
| **เช้า** | Authentication | ระบบสมาชิก, การ Login/Register |
| | Security | การป้องกัน XSS, CSRF, SQL Injection |
| **บ่าย** | CRUD Workshop | การสร้างระบบจัดการข้อมูลสมบูรณ์ |
| | Reporting | การสร้างรายงานและ Export ข้อมูล |

### 3. เครื่องมือที่จำเป็น (Prerequisites)

| เครื่องมือ | เวอร์ชัน | หน้าที่ |
|----------|---------|--------|
| **PHP** | 8.2+ | ภาษาโปรแกรมหลัก |
| **Composer** | 2.0+ | ตัวจัดการ Dependency |
| **Node.js & NPM** | 16+ | จัดการ Frontend Assets |
| **VS Code** | ล่าสุด | Code Editor พร้อม AI Extensions |
| **Laravel Installer** | ล่าสุด | เครื่องมือสร้างโปรเจกต์ Laravel |

#### คำสั่งตรวจสอบเวอร์ชัน

```bash
# ตรวจสอบเวอร์ชัน PHP
php -v

# ตรวจสอบเวอร์ชัน Composer
composer -V

# ตรวจสอบเวอร์ชัน Node.js
node -v
```

### 4. การใช้ AI ในการพัฒนา

AI ไม่ได้มาแทนที่โปรแกรมเมอร์ แต่โปรแกรมเมอร์ที่ใช้ AI จะมาแทนที่โปรแกรมเมอร์ที่ไม่ได้ใช้ ในหลักสูตรนี้เราจะเน้นการเขียน **Prompt** ที่มีประสิทธิภาพเพื่อ:

| การใช้งาน AI | ประโยชน์ |
|-------------|--------|
| ออกแบบฐานข้อมูล | สร้าง Schema และความสัมพันธ์ที่เหมาะสม |
| สร้าง Boilerplate Code | ลดเวลาการเขียนโค้ดพื้นฐานที่ซ้ำซ้อน |
| ตรวจสอบความปลอดภัย | ค้นหาช่องโหว่และแนะนำวิธีแก้ไข |
