AUTHORIZATION

ACL & Authorization

Managing User Permissions

Gates

Simple Closure based

Define Gate (AppServiceProvider):

Gate::define('update-post', function (User $user, Post $post) { return $user->id === $post->user_id; });

Check Gate:

if (Gate::allows('update-post', $post)) { // The user can update the post... }

Policies

Class based (Model focused)

Create Policy:

php artisan make:policy PostPolicy

Usage in Controller:

$this->authorize('update', $post);

Usage in Blade:

@can('update', $post) <button>Edit</button> @endcan
