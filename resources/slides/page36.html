Common Security Threats

OWASP Top 10 & Laravel Protection

SQL Injection

การแทรกคำสั่ง SQL ผ่าน Input เพื่อเจาะฐานข้อมูล

Risk:

ข้อมูลรั่วไหล, ถูกลบ, หรือแก้ไข

Laravel Fix:

// Eloquent uses bindings automatically User::where('email', $input)->first();

XSS (Cross-Site Scripting)

การฝัง Script อันตรายในหน้าเว็บ เพื่อขโมย Session หรือ Cookie

Risk:

Hijack session users

Laravel Fix:

// Blade escapes output by default {{ $user_input }}

CSRF

การหลอกให้ User ทำคำสั่งบางอย่างโดยไม่รู้ตัว

Risk:

Unwanted actions

Laravel Fix:

// Verify CSRF Token @csrf
