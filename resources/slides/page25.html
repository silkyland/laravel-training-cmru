Attribute Casting

การแปลง Data Types ของ Attributes อัตโนมัติ

Common Casts

ประเภทที่รองรับ

arraybooleancollectiondatedatetimedecimalfloatintegerjsonobjectstringtimestamp
protected $casts = [
  'is_admin' => 'boolean',
  'price' => 'decimal:2',
  'options' => 'json',
];

Date Casting

จัดการวันที่

Date Attributes:

protected $dates = [
  'created_at', 'deleted_at'
];

Date Format:

protected $dateFormat = 'd/m/Y H:i';

ผลลัพธ์: $user->created_at->format('d/m/Y')

Array & JSON

จัดการข้อมูลโครงสร้าง

Array Casting:

'options' => 'array'

เก็บเป็น PHP array

JSON Casting:

'options' => 'json'

เก็บเป็น JSON string

Usage: $user->options = ['key' => 'value'];

Custom Casts

สร้าง Cast Class

// php artisan make:cast Json
class JsonCast implements CastsAttributes
{
  public function get($model, $key, $value, $attr) {
    return json_decode($value, true);
  }
  public function set($model, $key, $value, $attr) {
    return json_encode($value);
  }
}

Enum Casts

PHP 8.1+ Enum

// Define Enum
enum UserStatus: string {
  case Active = 'active';
  case Inactive = 'inactive';
}
// In Model
protected $casts = [
  'status' => UserStatus::class,
];

Benefits

ข้อดีของ Casting

Automatic type conversion
Prevents invalid data types
Consistent data format
Better IDE support

Tip: ใช้ casting เสมอเพื่อ type safety
