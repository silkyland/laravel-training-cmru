Deployment Guide

Pushing to Production

Environment

ตั้งค่า .env ให้ถูกต้อง

APP_ENV=production APP_DEBUG=false

* ห้ามเปิด APP_DEBUG=true บน Production เด็ดขาด!

Optimization

รันคำสั่งเพื่อความเร็ว

  • composer install --optimize-autoloader --no-dev
  • php artisan config:cache
  • php artisan route:cache
  • php artisan view:cache

Permission

โฟลเดอร์ที่ต้องเขียนได้

  • storage/
  • bootstrap/cache/

Laravel Forge / Envoyer เป็น Tools ช่วย Deploy ที่ยอดเยี่ยม
