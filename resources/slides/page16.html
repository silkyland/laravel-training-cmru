ADVANCED ROUTING

Advanced Routing Features

Named Routes, Route Groups, Model Binding และ Middleware

Named Routes

ตั้งชื่อ Route เพื่อเรียกใช้ง่าย

route('profile') Easy URLs Parameters
Route::get('/user/profile', [UserProfileController::class, 'show'])->name('profile');
route('profile') route('profile', ['id' => 1])

Route Groups

จัดกลุ่ม Routes ที่มีคุณสมบัติเหมือนกัน

Middleware Prefix Namespace
Route::middleware(['auth'])->group(function () { Route::get('/dashboard', 'DashboardController@index'); Route::get('/profile', 'ProfileController@index'); });

Route Model Binding

Auto-resolve model จาก route parameters

Auto Resolution Type Hinting Clean Code
public function show(User $user) { return $user; } // /users/1 → Auto-resolves User::find(1)

Middleware in Routes

Apply middleware โดยตรงใน routes

Multiple Middleware Parameters Single Route
Route::get('/admin', 'AdminController@index') ->middleware('auth', 'role:admin');
