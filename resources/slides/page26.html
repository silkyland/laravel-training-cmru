ELOQUENT COLLECTIONS

Eloquent Collections

Wrapper สำหรับ array ของ models พร้อม methods จัดการข้อมูลที่ทรงพลัง

Common Methods

Methods ที่ใช้บ่อย

all() get() first() find() where() pluck()
map() filter() reject() each() groupBy()
sortBy() take() skip() count() isEmpty()

Example:

$users = User::all(); $names = $users->pluck('name'); $filtered = $users->filter(fn($u) => $u->age > 18);

Lazy vs Eager Loading

Lazy Loading (Default)

$users = User::all(); foreach($users as $user) { $user->orders; // Query ทุกรอบ }

Eager Loading (with())

$users = User::with('orders')->get(); foreach($users as $user) { $user->orders; // ไม่มี Query }

N+1 Problem

Problem:

Query ซ้ำๆ เมื่อ loop ผ่าน collection

Solution - Eager Loading:

// โหลด orders พร้อม users User::with('orders')->get(); // โหลดหลาย relationships User::with(['orders', 'profile', 'roles'])->get(); // โหลด nested relationships User::with('orders.items')->get();
