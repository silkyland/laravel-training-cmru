ENVIRONMENT SETUP

PHP Installation

ติดตั้ง PHP สำหรับแต่ละระบบปฏิบัติการ

Windows

ใช้ XAMPP/WAMP

1
Download from Apache Friends
2
Install with default settings
3
Enable extensions in php.ini
4
Restart Apache

Enable in php.ini:

extension=openssl
extension=pdo_mysql
extension=pdo_pgsql
extension=mbstring

macOS

ใช้ Homebrew

1
Install Homebrew ก่อน
2
Update Homebrew
3
Install PHP 8.2
4
Start PHP service

Run commands:

brew update
brew install php@8.2
brew services start php@8.2
brew install composer

Ubuntu/Debian

ใช้ APT Package Manager

1
Update package list
2
Install PHP 8.2 core
3
Install extensions
4
Install Composer

Run commands:

sudo apt update
sudo apt install php8.2
sudo apt install php8.2-mysql
sudo apt install composer

Verify Installation

ตรวจสอบว่าติดตั้งเรียบร้อย

php --version
php -m
