ELOQUENT RELATIONSHIPS

Eloquent Relationships - Part 1

One-to-One และ One-to-Many Relationships

One-to-One

1 ต่อ 1 Relationship

hasOne() belongsTo()

User Model:

public function profile() { return $this->hasOne(Profile::class); }

Profile Model:

public function user() { return $this->belongsTo(User::class); }
Usage:
$user->profile->phone
$profile->user->name

One-to-Many

1 ต่อ หลาย Relationship

hasMany() belongsTo()

User Model:

public function orders() { return $this->hasMany(Order::class); }

Order Model:

public function user() { return $this->belongsTo(User::class); }
Usage:
$user->orders->each(...)
$order->user->name

Lazy Loading

Default Behavior

โหลดข้อมูลเมื่อเข้าถึงเท่านั้น

$users = User::all();
foreach($users as $user) {
  $user->orders; // Query ทุกรอบ
}

Eager Loading

Prevent N+1 Problem

โหลด relationships พร้อมกัน

$users = User::with('orders')->get();
foreach($users as $user) {
  $user->orders; // ไม่มี Query เพิ่ม
}
