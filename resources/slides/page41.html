Caching Strategies

Speed Up Your Application

Application Cache

เก็บข้อมูลที่ใช้บ่อยไว้ใน Memory (Redis/Memcached)

// Remember for 60 minutes $users = Cache::remember('users', 3600, function () { return DB::table('users')->get(); });

Drivers: file (default), redis, memcached, database

Route & Config Cache

สำหรับ Production Environment เท่านั้น

php artisan config:cache

รวมไฟล์ config เป็นไฟล์เดียว

php artisan route:cache

แปลง Route ทั้งหมดเป็น Array Map (เร็วมาก)
