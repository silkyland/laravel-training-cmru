Encryption vs Hashing

Encryption

การเข้ารหัสแบบย้อนกลับได้

Two-way (Decryptable)

ใช้สำหรับข้อมูลที่ต้องการอ่านกลับมาได้ เช่น เลขบัตร, ข้อมูลส่วนตัว, Token ต่างๆ

use Illuminate\Support\Facades\Crypt; $encrypted = Crypt::encryptString('secret'); $decrypted = Crypt::decryptString($encrypted);

* Laravel ใช้ OpenSSL และ AES-256-CBC

Hashing

การเข้ารหัสแบบทางเดียว

One-way (Irreversible)

ใช้สำหรับตรวจสอบความถูกต้อง แต่ไม่ต้องการรู้ค่าจริง เช่น รหัสผ่าน

use Illuminate\Support\Facades\Hash; // Store $hashed = Hash::make('password123'); // Verify if (Hash::check('password123', $hashed)) { // Correct }
