CSRF Protection

ความปลอดภัยของฟอร์ม

Laravel ปกป้องเว็บจาก Cross-Site Request Forgery (CSRF) โดยอัตโนมัติ ทุกฟอร์มที่ส่ง method POST, PUT, DELETE ต้องมี CSRF Token

Required: ใช้ @csrf ในทุกฟอร์ม

Standard Form:

<form method="POST" action="/profile"> @csrf ... </form>

Spoofing Methods (PUT, DELETE):

<form method="POST" action="/post/1"> @csrf @method('DELETE') <button>Delete</button> </form>

HTML Form รองรับแค่ GET/POST จึงต้องใช้ @method ช่วย
