SYSTEM DEVELOPMENT

Building CRUD System

Create, Read, Update, Delete

Resource Controller

สร้าง Controller เดียว ครบทุกฟังก์ชัน CRUD ตามมาตรฐาน RESTful

php artisan make:controller PostController --resource
index() : List all items
create() : Show create form
store() : Save new item
show() : Show item detail
edit() : Show edit form
update() : Save changes
destroy() : Delete item

Route Model Binding

Laravel ช่วยดึงข้อมูลจาก Database อัตโนมัติตาม ID ใน URL

Implicit Binding:

// Route definition Route::get('/posts/{post}', ...); // Controller Method public function show(Post $post) { // $post is already fetched from DB! return view('posts.show', compact('post')); }

* ช่วยลดโค้ด User::find($id) และ 404 check
