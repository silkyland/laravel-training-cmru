Background Processing

Why Queues?

ย้ายงานที่ใช้เวลานานไปทำเบื้องหลัง (Asynchronous) เพื่อไม่ให้ User ต้องรอนาน

  • ส่งอีเมล (Email Sending)
  • สร้างรายงาน PDF (Report Generation)
  • ประมวลผลรูปภาพ/วิดีโอ (Image Processing)
  • เรียก API ภายนอก (External API Calls)

Dispatching Job:

ProcessPodcast::dispatch($podcast);

Running Worker:

php artisan queue:work

Job Class Structure:

public function handle() { // Long running task... }
