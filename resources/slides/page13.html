Service Container & Dependency Injection

หัวใจของ Laravel: การจัดการ Dependencies อย่างมีประสิทธิภาพ

Service Container

IoC Container

  • จัดการ dependencies และ class instances
  • หัวใจของ Laravel Framework
  • คลาส singleton ที่จัดการทุกการ binding
  • Resolve dependencies อัตโนมัติ

Dependency Injection

DI Pattern

  • แทรก dependencies ผ่าน constructor/method
  • ลด coupling ระหว่าง components
  • ทำให้ code ทดสอบได้ง่าย
  • ยืดหยุ่นและบำรุงรักษาง่าย

Binding Methods

bind() singleton() instance() method()

Benefits

ทดสอบง่าย (Testable)
จัดการ dependencies ดี (Manageable)
Code ยืดหยุ่น (Flexible)
