ติดตั้ง Composer และสร้างโปรเจกต์ Laravel แรก

Composer Installation

Download from getcomposer.org
Run installer wizard
Verify: composer --version

Create Laravel Project

composer create-project laravel/laravel my-project
cd my-project
php artisan serve

Development Server

php artisan serve

Access at http://localhost:8000

.env Configuration

APP_NAME "My App"
APP_ENV "local"
APP_URL "http://localhost:8000"
DB_DATABASE "laravel"
DB_USERNAME "root"
DB_PASSWORD ""
