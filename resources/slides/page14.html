How Facades Work

// Facade Syntax (Static Interface)
DB::table('users')->get();
// Behind the Scenes (Service Container)
$app['db']->table('users')->get();

Definition

  • Static interface สำหรับ services
  • เข้าถึง service container
  • ทำให้ code กระชับ
