How Facades Work

// Facade Syntax (Static)
DB::table('users')->get();
// Behind the Scenes
$app['db']->table('users')->get();

Facade Flow

1. Static Call

DB::table('users')

2. __callStatic()

Resolve from Container

3. Service Instance

$app['db']->table()

Definition

  • Static interface สำหรับ services ใน Container
  • เข้าถึง service container ได้ง่าย
  • ทำให้ code กระชับ อ่านง่าย
  • ยังคง testable ด้วย Mockery

Benefits

สะดวกในการใช้งาน
Code กระชับ สั้น
เข้าถึง services ง่าย
รองรับ Unit Testing

Tip: ใช้ Facade เมื่อต้องการ syntax สั้น แต่ใช้ DI (Dependency Injection) เมื่อต้องการ testability สูง
