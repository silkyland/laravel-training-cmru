Accessors

Getter Methods

Accessor

Naming Convention:

get{Attribute}Attribute // Example: getFullNameAttribute

Example Implementation:

public function getFullNameAttribute() { return $this->first_name . ' ' . $this->last_name; }

Usage:

$user->full_name

Mutators

Setter Methods

Mutator

Naming Convention:

set{Attribute}NameAttribute // Example: setFirstNameAttribute

Example Implementation:

public function setFirstNameAttribute($value) { $this->attributes['first_name'] = ucfirst($value); }

Usage:

$user->first_name = 'john';
