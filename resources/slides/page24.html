Getter และ Setter Methods สำหรับ Manipulate ข้อมูล

Accessors

Getter Methods

Convention:

get{Attribute}Attribute
// getFullNameAttribute

Example:

public function getFullNameAttribute() {
  return $this->first_name . ' ' .
    $this->last_name;
}

Usage: $user->full_name

Mutators

Setter Methods

Convention:

set{Attribute}Attribute
// setFirstNameAttribute

Example:

public function setFirstNameAttribute($v) {
  $this->attributes['first_name']
    = ucfirst($v);
}

Usage: $user->first_name = 'john';

Laravel 11+ Syntax

Attribute Class (ใหม่)

New Way:

use Illuminate\Database\Eloquent\
  Casts\Attribute;

protected function firstName(): Attribute {
  return Attribute::make(
    get: fn($v) => ucfirst($v),
    set: fn($v) => strtolower($v),
  );
}

รวม get/set ไว้ที่เดียว
