VIEWS & BLADE

Views & Blade - Basics

Template Engine ที่ทรงพลังและใช้ง่ายของ Laravel

Echo Statements

แสดงข้อมูลใน View

Display Variable:

{{ $name }}

With Default:

{{ $name ?? 'Guest' }}

Raw HTML:

{!! $html !!}

Control Structures

การควบคุม Flow

@if($condition) True @elseif($other) Other @else False @endif
@for($i = 0; $i < 10; $i++) Item {{ $i }} @endfor

Loops

วนลูป Data

Foreach:

@foreach($users as $user) {{ $user->name }} @endforeach

Foreach with Key:

@foreach($users as $key => $user) {{ $key }}: {{ $user->name }} @endforeach

Blade Files

โครงสร้างไฟล์

อยู่ใน resources/views/
ใช้นามสกุล .blade.php
Supports nesting directories
Compiled และ cached

Returning Views

ส่ง View จาก Controller

Simple View:

return view('welcome');

With Data:

return view('users.index', compact('users'));

With Array:

return view('profile', ['user' => $user]);
