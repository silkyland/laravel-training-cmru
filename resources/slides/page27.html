VIEWS & BLADE

Views & Blade Basics

Template Engine ที่ทรงพลังและใช้ง่ายของ Laravel

Echo Statements

แสดงข้อมูลใน View

Display Variable:

{{ $name }}

With Default:

{{ $name ?? 'Guest' }}

Raw HTML (unescaped):

{!! $html !!}

Control Structures

การควบคุม Flow

@if($user->isAdmin)
  Admin Panel
@elseif($user->isMod)
  Mod Panel
@else
  User Panel
@endif
@unless($user->banned)
  Welcome!
@endunless

Loops

วนลูป Data

@foreach($users as $user)
  {{ $user->name }}
@endforeach
@forelse($users as $user)
  {{ $user->name }}
@empty
  No users found.
@endforelse

Blade Files

โครงสร้างไฟล์

อยู่ใน resources/views/
นามสกุล .blade.php
Supports nesting directories
Compiled & cached อัตโนมัติ

Dot notation: users.index → views/users/index.blade.php

Returning Views

ส่ง View จาก Controller

Simple:

return view('welcome');

With compact():

return view('users.index', compact('users'));

With Array:

return view('profile', ['user' => $user]);

$loop Variable

ข้อมูลใน Loop

@foreach($items as $item)
  {{ $loop->index }}
  {{ $loop->first }}
  {{ $loop->last }}
  {{ $loop->count }}
  {{ $loop->remaining }}
@endforeach
