ELOQUENT QUERY SCOPES

Eloquent Query Scopes

Method สำหรับเพิ่มเงื่อนไขให้กับ Query อย่าง Reuse ได้

Local Scopes

Scope ปกติ

Local

Defining:

public function scopeActive($query) { return $query->where('active', 1); }

Usage:

User::active()->get(); User::active()->where('age', '>', 18)->get();

Global Scopes

Scope ทั่วไป

Global

Defining:

protected static function booted() { static::addGlobalScope('age', function (Builder $builder) { $builder->where('age', '>', 18); }); }

ผล: ใช้เงื่อนไปกับทุก query อัตโนมัติ

Dynamic Scopes

Scope รับ Parameters

Dynamic

Defining:

public function scopeOfAge($query, $age) { return $query->where('age', '>', $age); }

Usage:

User::ofAge(18)->get(); User::ofAge(25)->get();

Benefits

Reuse logic ได้

Code กระชับ

แยก logic ออกจาก controller

Easy to test
