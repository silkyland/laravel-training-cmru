# ระบบจองห้องประชุม (Meeting Room Booking)

> Prompt สำหรับสร้างระบบจองห้องประชุมมหาวิทยาลัย ด้วย Laravel

---

## Prompt 1: วางแผนโครงสร้างทั้งระบบ

```
สร้างระบบจองห้องประชุมสำหรับมหาวิทยาลัย ด้วย Laravel 11, MySQL, Blade + TailwindCSS

ข้อมูลที่ต้องจัดการ:
- ห้องประชุม: ชื่อห้อง, อาคาร, ชั้น, ความจุ (คน), อุปกรณ์ (โปรเจกเตอร์/ไมค์/กระดาน), รูปภาพ, สถานะ (พร้อมใช้/ปิดปรับปรุง)
- การจอง: ผู้จอง, ห้อง, วันที่, เวลาเริ่ม, เวลาสิ้นสุด, หัวข้อการประชุม, จำนวนผู้เข้าร่วม, หมายเหตุ, สถานะ (รออนุมัติ/อนุมัติ/ไม่อนุมัติ/ยกเลิก)
- ผู้ใช้: บุคลากร (จองได้), ผู้ดูแลระบบ (อนุมัติ/จัดการห้อง)

ฟีเจอร์:
1. CRUD ห้องประชุม (Admin)
2. ปฏิทินแสดงการจองรายวัน/รายสัปดาห์
3. ฟอร์มจองห้อง พร้อมตรวจสอบว่าห้องว่างหรือไม่ (ไม่ให้จองซ้อนกัน)
4. ระบบอนุมัติการจอง (Admin อนุมัติ/ไม่อนุมัติ พร้อมเหตุผล)
5. แจ้งเตือนผ่าน Email เมื่อสถานะเปลี่ยน
6. ค้นหาห้องว่างตามวันที่ + เวลา + ความจุ
7. รายงานสถิติการใช้ห้องประชุม (ห้องไหนใช้บ่อย, ช่วงเวลายอดนิยม)
8. ประวัติการจองของผู้ใช้แต่ละคน

ให้สร้าง:
1. Migration ทุกตาราง พร้อม Foreign Keys
2. Models พร้อม Relationships, Scopes, Accessors
3. Resource Controller: RoomController, BookingController
4. Form Requests: StoreBookingRequest (ตรวจเวลาซ้อน)
5. Blade Views: ปฏิทิน, ฟอร์มจอง, รายการจอง, หน้าอนุมัติ
6. Middleware: CheckRole (admin/staff)
7. Routes: web.php แบ่ง admin กับ staff
8. Seeder: ห้องประชุมตัวอย่าง 10 ห้อง
```

## Prompt 2: ตรวจสอบเวลาจองซ้อน

```
สร้าง Validation Rule ใน Laravel สำหรับตรวจสอบว่าห้องประชุมถูกจองซ้อนเวลาหรือไม่

เงื่อนไข:
- ตรวจจาก room_id, date, start_time, end_time
- ต้องไม่ซ้อนกับการจองที่สถานะเป็น "อนุมัติ" หรือ "รออนุมัติ"
- กรณี Update ต้องยกเว้นการจองตัวเอง
- แสดง Error Message ภาษาไทย: "ห้องนี้ถูกจองในช่วงเวลาดังกล่าวแล้ว"

ให้สร้างเป็น Custom Rule class
```

## Prompt 3: ปฏิทินแสดงการจอง

```
สร้าง Blade Template แสดงปฏิทินการจองห้องประชุมรายสัปดาห์:
- แกน X: วันจันทร์ - วันศุกร์
- แกน Y: เวลา 08:00 - 17:00 (แบ่งทีละ 1 ชั่วโมง)
- แสดงการจองเป็นบล็อกสีตามสถานะ (เขียว=อนุมัติ, เหลือง=รออนุมัติ, แดง=ไม่อนุมัติ)
- คลิกช่องว่างเพื่อจองได้
- มีปุ่มเลื่อนสัปดาห์ ก่อนหน้า/ถัดไป
- ใช้ TailwindCSS, ไม่ต้องใช้ JavaScript Library ภายนอก
- Controller ส่งข้อมูล bookings ของสัปดาห์ที่เลือก
```
