สร้าง layout เว็บข่าวโดยมีรายละเอียดดังนี้
- logo, 
- main menu (หน้าแรก, ข่าวสาร, เกี่ยวกับ CMRU,ติดต่อเรา) 
- footer 
เอาแค่หน้า view ก่อนโดยใช้ bootstrap css 

- สร้าง route / และสร้าง view หน้าแรกเป็นรูปแบบข่าวสารให้หน่อย 

ออกแบบฐานข้อมูลระบบข่าวสารโดยข่าวสารจะมีหมวดหมู่ข่าวสาร ผู้เขียนข่าว เขียน model และ migration file ก็พอ

ออกแบบฐานข้อมูลเพื่อเก็บ banner โดย banner จะมีวันหมดอายุ และสามารถเรียงลำดับได้ run migrate ให้เลย

เขียนข่าวตัวอย่างเข้าไปในฐานมูล จำนวน 10 รายการเป็นเรื่อง

สร้างรูปภาพเก็บไว้ใน โฟล์เดอร์ uploads จำนวน 5 รายการเป็นรูป banners และบันทึกข้อมูลลงฐานข้อมูล banners ให้ด้วย

ทำหน้า admin dashboard ให้สวยงามเพื่อจัดการ user ให้ก่อนโดยเขียน Route, Controller, model, และ หน้า view 
- ทำ share layout สำหรับ admin โดยเฉพาะ
ที่หน้า admin จะมี prefix ของ route เป็น /admin เสมอ 
อย่าพึ่งใส่ระบบ authen เพราะกำลังทำความเข้าใจ Laravel 
ทำหน้าจัดการหมวดหมู่ข่าวให้หน่อย
ทำ banner ต่อเลย
ทำหน้าจัดการข่าวสารโดยใช้ wysiwyg เพื่อจัดการ content สามารถอัพโหลดเพื่อแทรกรูปภาพ และเครื่องมืออื่นๆที่จำเป็น


ทำระบบ basic (email, password) authentication โดยมีหน้า login แล้วตกแต่งให้สวยงาม หน้า admin ทั้งหมดจะถูกกั้นสิทธิ์ไว้ไม่ให้เข้าถึงถ้ายังไม่ login 

อยากให้เพิ่มการจำกัดสิทธิ์การเข้าถึงโดย admin จะสามารถเข้าถึงได้ทุกเมนู แต่ writer มีสิทธิ์ได้แค่เข้าถึงการแก้ไข banner และ ข่าวสารเท่านั้น โดย user บัณฑิต เป็น admin และ สมชาย เป็น Writer ถ้าสร้าง user ใหม่ในหน้า admin จะ default เป็น writer เพิ่ม migration และ run ให้เลย 

ให้ทำหน้าแรกเพื่อดึงข้อมูลข่าวสารและ banner จากข้อมูลจริงในฐานข้อมูล และแก้ไขเมนูให้สอดคล้องกับฐานข้อมูล ทำหน้าติดต่อฉัน เกี่ยวกับฉันให้ด้วย เพื่อสำหรับ user ทั่วไปที่เข้ามาดูเว็บไซต์ของเราด้วย ตกแต่งให้สวยงาม
